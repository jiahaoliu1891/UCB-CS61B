## Java 基础

* 每个 *[filename].java* 文件里定义一个 class
  * Filename 必须与 class 同名
  * class的初始化函数 与 class 同名
* 每一个 class 可以定义**成员**和**方法**
  * 方法分为 **静态方法** 和 **实例方法**
  * 静态方法：
    * 使用 classname.method() 调用，如： Math.sqrt()
    * 如果一个方法==与他所在类的实例对象无关==，那么它就应该是**静态**的
    * 静态方法只能访问静态成员
  * 动态方法
    * 先实例化对象，再调用
    * 若==与实例有关==，那么这个方法就应该是**实例**的
